# Hearo AI — Figma Design & Prototype

This directory provides access to the Figma design file for **Hearo AI**, an AI-driven mobile application focused on improving communication accessibility for Deaf and hard-of-hearing individuals.

## 📱 Project Overview

The Figma design includes the visual interface and interaction design developed for the Hearo AI mobile application.

Key areas of the project may include:

* Mobile application interface
* User onboarding
* Main navigation
* AI-powered communication features
* Accessibility-focused interactions
* Sign language-related features
* Interactive user flows
* High-fidelity UI design
