# Hearo AI — AI Communication Bridge for the Deaf

Hearo AI is an AI-driven mobile application designed to improve communication accessibility for Deaf and hard-of-hearing individuals. The application combines artificial intelligence with inclusive user experience design to help bridge communication gaps in everyday interactions.

## 📌 Project Overview

Hearo AI was developed as a UI/UX design project focused on accessibility and inclusive technology. The project explores how AI can support communication between Deaf individuals and the wider community through an intuitive and user-friendly mobile experience.

## 🎯 Problem Statement

Deaf and hard-of-hearing individuals often face communication barriers in daily activities, especially when interacting with people who do not understand sign language. Existing communication methods may not always be fast, accessible, or practical in real-time situations.

Hearo AI aims to provide a more inclusive digital solution by integrating AI-powered communication tools into a single mobile application.

## 💡 Our Solution

Hearo AI provides several core features:

* 🗣️ **Speech-to-Text** — Converts spoken language into readable text in real time.
* 🤟 **Text-to-Sign Language** — Helps translate text into visual sign language communication.
* 🤖 **AI Assistant** — Supports users with contextual communication assistance.
* 📚 **Sign Language Learning** — Provides an interactive space for learning sign language.
* ♿ **Accessible User Experience** — Designed with simplicity, readability, and inclusive interaction in mind.

## 🧠 Design Process

The project follows a Design Thinking approach:

1. **Empathize** — Understanding communication challenges experienced by Deaf users.
2. **Define** — Identifying key accessibility and communication problems.
3. **Ideate** — Exploring AI-based solutions and inclusive interaction concepts.
4. **Prototype** — Designing wireframes and high-fidelity mobile interfaces in Figma.
5. **Test** — Evaluating usability and improving the design based on feedback.
